# Types agrégés

Les types agrégés (collections) permettent de stocker et manipuler plusieurs valeurs au sein d'une même structure.

Python propose quatre types agrégés principaux:
- Les listes : collections ordonnées et modifiables
- Les dictionnaires: collections de paires clé-valeurs.
- Les ensembles (sets) : collections non ordonnées d'éléments uniques.
- Les tuples : collections ordonnées et immuables
